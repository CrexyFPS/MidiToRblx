# MidiToRblx
!!! PLEASE DELETE THE TEMPLATE MIDI INSIDE OF THE "MIDIs" Folder !!!

TO USE ↓↓↓↓

To use simply drag ur midi file into the folder called MIDIs.
There may not be any other midis but the one u want to convert.
